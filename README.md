## TV Series Rating and Description Website

This project is designed for a website where users can rate the TV series they watch and describe them in one word to create and share their lists.

## Features

Users can create an account and log in.
Users can search for TV series and rate them.
Users can describe each TV series in one word.
Users can create and share their lists.
Users can view other users' lists.

## Installation

Clone or download this repository.
Open a terminal or command prompt and go to the project folder.
Run `npm install` to install the required packages.
Run `npm start` to start the application.

## Usage

Start the application.
Create an account or log in.
Search for TV series and rate them.
Describe each TV series in one word.
Create and share your lists.

This project aims to help users explore their preferences and discover new TV series recommendations.
